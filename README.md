# ULTRAPONG
Networked game for a Continuous Assignment 2 of our Multiplayer Distributed Programming module.
Ever wanted to play a pong that was not boring?
YOU CAN! AND WITH UP TO 20 FRIENDS ON LOCAL NETWORK NOW!

### Controls (can rebind)
Move - W/S/A/D
Use powerup - Space

### Powerups
When bouncing a ball into a powerup you will pick it up.
Blue - you dash (catch that ball!)
Yellow - change direction of ball horizontally (very tricky)
Red - change direction of the ball vertically
Purple - slow down the ball
